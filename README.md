# Smart College Admission FAQ Bot

A Python chatbot that answers frequently asked college admission questions using NLP techniques.

## Features

- Modern CustomTkinter UI
- FAQ-based chatbot
- TF-IDF Vectorization
- Cosine Similarity Matching
- Dark Mode Interface
- Confidence-based Answer Retrieval

## Technologies Used

- Python
- CustomTkinter
- NLTK
- Scikit-Learn

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
